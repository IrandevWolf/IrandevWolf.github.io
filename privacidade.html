Política de Privacidade

Esta política de privacidade descreve como o aplicativo Iran Dev coleta, usa e protege as informações dos usuários.

Coleta de Informações

O aplicativo pode coletar dados anônimos para fins de análise e melhoria da experiência do usuário. Nenhuma informação pessoal identificável é coletada sem consentimento.

Uso das Informações

As informações coletadas são utilizadas exclusivamente para melhorar o desempenho e funcionalidades do aplicativo.

Compartilhamento de Dados

Não compartilhamos dados dos usuários com terceiros, exceto quando exigido por lei.

Publicidade

O aplicativo pode exibir anúncios fornecidos pelo Google AdMob. Esses anúncios podem utilizar cookies para personalização.

Segurança

Tomamos medidas para proteger os dados dos usuários, mas nenhuma transmissão pela internet é 100% segura.

Alterações

Esta política pode ser atualizada periodicamente. Recomendamos que o usuário revise esta página regularmente.
